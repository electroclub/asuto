СИСТЕМА УПРАВЛЕНИЯ ПОСТАНОВОЧНЫМ ОСВЕЩЕНИЕМ СЦЕНЫ НА БАЗЕ КОМПЬЮТЕРИЗОВАНОГО ПУЛЬТА

НАЗНАЧЕНИЕ И СТРУКТУРНАЯ СХЕМА.

Система управления постановочным освещением сцены на базе компютеризованого пульта (далее - система) предназначенная для ввода, хранения и воспроизведения световой партитуры, а также управления диммерными блоками по интерфейсу типа DMX512.

Структурная схема системы приведена на рисунке 1.

Рисунок 1. Структурная схема системы.

В состав системы входят:

  • Персональный компьютер с принтером, блоком бесперебойного питания и световым редактором;
  • Пульт управления театральным освещением;
  • Двунаправленный специализированный адаптер интерфейса (USB - DMX512);
  • Комплект кабелей.

Внешний вид системы приведен на рис.2.


Рисунок 2. Внешний вид системы

Назначение составных частей системы.

Персональный компьютер со световым редактором предназначен для ввода, хранения и воспроизведения световой партитуры. Принтер предназначен для вывода на бумажный носитель текстовой документации (партитура, комментарий, протокол ведения спектакля). Блок бесперебойного питания предназначен для обеспечения электропитанием составных частей системы с целью исключения возможности выхода их из строя при краткосрочном пропадании напряжения в электросети.

Пульт управления театральным освещением предназначен для ввода параметров управления источниками света и передачи их в персональный компьютер по интерфейсу типа DMX 512.

Двухнаправленный специализированный адаптер интерфейса (USB - DMX512) предназначен для приема от пульта и передачи в димерные блоки информации по интерфейсу DMX512, обмена информацией по интерфейсу типа USB с персональным компьютером.

Комплект кабелей предназначен для обмена информацией между составными частями системы, подачи напряжения питания на элементы системы.

ПОДКЛЮЧЕНИЕ

Подключение системы осуществляется следующим образом:

  • Подключите составные части компьютера согласно документации на компьютер.
  • Подключите принтер к компьютеру согласно документации на принтер.
  • Подключите выход "DMX 512 OUT" пульта ко входу "DMX 512 IN" адаптера.
  • Подключите выход "DMX 512 OUT" адаптера к кабелю интерфейса "DMX 512 IN", который с другой стороны подключен к димерным блокам.
  • Подключите выход "USB" компьютера ко входу "USB" адаптера.
  • Подключите входы питания "~220В, 50 Гц" компьютера, принтера и пульта к выходам питания "~220В, 50 Гц" блока бесперебойного питания.
  • Подключите вход питания "~220В,50 Гц" блока бесперебойного питания к сети питания "~220В, 50 Гц".

ВКЛЮЧЕНИЕ/ВЫКЛЮЧЕНИЕ СИСТЕМЫ.

Включение системы осуществляется в следующей последовательности:

  • Включите питание блока бесперебойного питания соответственно документации;
  • Включите питание персонального компьютера;
  • Включите питание пульта (при необходимости);
  • Включите питание принтера (при необходимости).

Выключение системы осуществляется в следующей последовательности:

  • Выключите питание принтера;
  • Выключите питание пульта;
  • Выключите питание персонального компьютера;
  • Выключите питание блока бесперебойного питания.

РЕЖИМЫ РАБОТЫ.

В зависимости от потребности система работает в трех основных режимах:

  • 1 - Введение и выполнение партитуры из серийного пульта и светового редактора компьютера;
  • 2 - Выполнение партитуры только из серийного пульта (BYPASS) с введением и формированием партитуры в световом редакторе компьютера;
  • 3 - Введение и выполнение партитуры только со светового редактора компьютера.

В режиме 1 информация о состоянии регуляторов по интерфейсу "DMX 512" поступает из внешнего пульта в компьютер. При этом световой редактор по командам оператора в реальном масштабе времени интерпретирует ее как состояние 1-48, 49-96..и т.д. каналов регулирования, или состояние плавных регуляторов управления партитурой (до 48) - групповых, общего, плавных регуляторов управления переходами между картинами вперед и назад по партитуре, темпу выполнения партитуры. Причем при количестве входных каналов от пульта - до 48, количество выходных регулируемых каналов может достигать 512 при необходимости.

Световой редактор осуществляет прием информации от пульта, автоматическое формирование, хранение и распечатку световой партитуры.

В режиме 2 выполнение партитуры осуществляется непосредственно из внешнего пульта, но с введением и формированием партитуры в световом редакторе. При этом информация передается из входа "DMX 512 IN" на выход "DMX 512 OUT" адаптера, параллельно поступая в компьютер для формирования партитуры.

В режиме 3 выполнение партитуры осуществляется со светового редактора без обязательного наличия внешнего пульта.

Система управления светом сцены обеспечивает:

  • формирование, хранение, модификацию и выполнение световых партитур спектаклей;
  • динамику изменения картин;
  • создание и выполнение световых эффектов;
  • оформление документации на спектакль (партитура, комментарий, план расположения источников света и протокол ведения спектакля).

Технические характеристики

Технические характеристики системы определяются техническими характеристиками составных частей системы, которые приведены в соответствующей документации.
